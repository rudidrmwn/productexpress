PRODUCT EXPRESS is application using MERN (MySQL Expressjs Reactjs Nodejs)

# Backend
### Installing 
```
cd backend
npm install
```


### Setup Environment Variable
```
DATABASE_NAME = 'product_express'
DATABASE_USERNAME = 'root'
DATABASE_PASSWORD = ''
```

### Running Backend
```
cd backend/nodemon index.js
```

# Frontend
### Installing 
```
npm install
```
### Running Frontend
```
npm start
```
